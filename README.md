
# Interior Design Studio Website

A modern, minimalistic website created for a small interior design studio Rêver.  
The site focuses on visual storytelling, clean layout, and smooth user experience, with emphasis on high-quality interior imagery.


##  Overview

This project is a responsive, image-centric website designed to showcase interior design projects in an elegant and unobtrusive way.  
The visual language is minimal, using neutral tones and subtle animations to allow the interiors themselves to take center stage.

The website is optimized for performance and usability across all devices and will be published online soon.

##  Design & Aesthetic

- Minimalistic design approach
- Neutral color palette (beige and brown tones)
- Clean typography and generous spacing
- Subtle, smooth animations and transitions
- Strong visual emphasis on interior photography


## Features


- **Responsive layout** – fully optimized for desktop, tablet, and mobile devices
- **Optimized images** for fast loading and improved performance
- **Project-based galleries** with:
  - Individual galleries per project
  - Responsive layout
  - Image captions
  - Pagination
  - Autoplay functionality
- Smooth page transitions and minimal animations
- Simple and intuitive navigation

## Installation

1. Clone the repository:

```bash
  git clone https://github.com/lory497/Interior-design-studio.git
```

2. Navigate to the project directory:

```bash
cd Interior-design-studio

```

3. Install dependencies (if applicable):


```bash
npm install

```
